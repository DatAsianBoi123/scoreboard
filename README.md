# Scoreboard
A scoreboard website designed to be used with FIRST robotics competitions.

## Features
- Hosting game sessions
- Creating custom games
- Viewing games
- Reffing games

## Locally Hosting

1. [Install rust](https://rust-lang.org/tools/install).
1. `clone` this repository and navigate to its root folder.
1. Execute `cargo run` and wait for the project to compile and run. Note that running this will expose the server to your local network.
1. Finally, navigate to [127.0.0.1:3000](http://127.0.0.1:3000) on your favorite web browser and begin hosting games!

To connect other devices on the same network, navigate to your local device's IP address instead. On Linux, this can be found using `ip route`.
